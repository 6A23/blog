# 八大行星闯关战

一个浏览器即可运行的八大行星主题闯关小游戏。

## 玩法

- 八大行星对应 8 个 2D 横版关卡。
- 每关有 3 波独有怪物和 1 个 Boss。
- 击败怪物或 Boss 可获得星币。
- 星币可用于升级武器或购买治疗包。
- 必须打赢当前行星 Boss，才会解锁下一关。
- 每进入下一关，怪物生命、移动速度和伤害都会继续提升。
- 每打赢一个行星 Boss，人物外形都会进化，看起来更强大。
- 每打赢一个行星 Boss，会获得一枚独特勋章并加入勋章收藏。


- 操作：A / D 或 ← / → 移动，W / ↑ 跳跃躲避，空格 / J 普通射击，K 蓄能炮。




## 运行

方式一：直接用浏览器打开 `index.html`。

方式二：在当前文件夹运行：

```bash
node server.js
```

然后打开：

```text
http://127.0.0.1:5173/index.html
```
