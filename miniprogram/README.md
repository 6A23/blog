# 🌍 地球指挥中心 · 微信小程序端

调用 Vercel 部署的 `/api/earth` 接口，展示地球指挥中心完整功能。

## 使用方法

1. 打开**微信开发者工具**
2. 点击「导入项目」，选择本目录 `miniprogram/`
3. AppID 填写你自己的小程序 AppID（或使用测试号）
4. 在 `app.js` 中确认 `apiBase` 地址正确：
   ```js
   apiBase: 'https://blog-6a23.vercel.app/api/earth'
   ```
5. 在微信开发者工具 → 详情 → 本地设置，勾选 **「不校验合法域名」**（开发阶段）

> 正式上线时需在微信公众平台 → 开发 → 开发管理 → 开发设置中，将 Vercel 域名加入 **request 合法域名**。

## 页面结构

| 页面 | 功能 |
|------|------|
| 首页（总览）| 威胁等级、地球状态摘要、月球基地摘要、最近事件 |
| 地球 | 状态 / 威胁 / 设施控制 / 事件日志 |
| 月球 | 基地总览 / 机器人状态 / 建造进度 / 材料库存 / 汇报&申请操作 |
